## 🔧 1. Formspree（メール送信機能）のセットアップ

### 目的
お問い合わせフォームから直接メールを受け取るためのサービスです。

### セットアップ手順

#### ステップ1: Formspreeに登録
1. https://formspree.io にアクセス
2. 「Sign Up」をクリック
3. メールアドレスを入力して登録
4. メール認証を完了

#### ステップ2: 新しいフォームを作成
1. ダッシュボードで「New Form」をクリック
2. フォーム名を入力（例：Contact Form）
3. 「Create」をクリック
4. **フォームID（例：f_abc123def456）** をコピー

#### ステップ3: HTMLに設定
1. `index.html`を開く
2. 以下の行を探す：
   ```
   <form class="contact-form" id="contactForm" action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```
3. `YOUR_FORM_ID` を手順2でコピーしたIDに置き換え
   例：`https://formspree.io/f/f_abc123def456`

#### ステップ4: テスト送信
1. ホームページを開く
2. お問い合わせフォームに入力
3. 「送信する」ボタンをクリック
4. 登録したメールアドレスに確認メールが届くか確認

✅ **メール送信機能が有効になります！**

## 📊 2. Google Analytics（アクセス解析）のセットアップ

### 目的
訪問者数、ページ閲覧数、ユーザー行動などを分析できます。

### セットアップ手順

#### ステップ1: Google Analyticsに登録
1. https://analytics.google.com にアクセス
2. Googleアカウントでログイン（なければ作成）
3. 「今すぐ開始」をクリック

#### ステップ2: プロパティを作成
1. アカウント名を入力（例：solace）
2. 「次へ」をクリック
3. プロパティ名を入力（例：solace-website）
4. タイムゾーンを「日本」に設定
5. 「作成」をクリック

#### ステップ3: ウェブストリームを設定
1. ウェブストリーム作成画面で：
   - ウェブサイトURL：あなたのサイトのURL
   - ストリーム名：solace
2. 「ストリームを作成」をクリック
3. **測定ID（G-xxxxxxxxxx）** をコピー

#### ステップ4: HTMLに設定
1. `index.html`を開く
2. `<head>`タグ内で以下を探す：
   ```
   <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
   ```
3. 両箇所の `G-XXXXXXXXXX` を手順3でコピーしたIDに置き換え

#### ステップ5: 確認
1. ホームページにアクセス
2. Google Analytics ダッシュボードで「リアルタイム」を確認
3. 自分のアクセスが表示されるか確認

✅ **アクセス解析が有効になります！**

### 便利な分析機能
- **リアルタイムレポート**：今この瞬間のアクセス状況
- **ユーザー**：訪問者の地域、デバイス、ブラウザ情報
- **イベント**：フォーム送信、ボタンクリックなどの行動追跡
- **ページ**：どのページがよく見られているか

## 💬 3. AIチャットボット（FAQ）

### 特徴
- ✅ 自動でよくある質問に答える
- ✅ キーワード検索で対応
- ✅ カスタマイズ可能なFAQデータベース
- ✅ 24時間稼働

### 使い方
1. ホームページ右下に吹き出しアイコン💬が表示されます
2. クリックするとチャットボットが開きます
3. 質問を入力すると、自動で回答が返ってきます

### FAQ内容をカスタマイズする方法

`script.js`を開いて、以下の部分を編集します：

```javascript
this.faqDatabase = [
    {
        keywords: ['納期', 'いつ', 'どのくらい'],  // ← 検索キーワード
        answer: '通常、シンプルなイラストは2～3週間...'  // ← 回答文
    },
    // 以下同様に追加可能
];
```

**例：新しいFAQを追加する場合**
```javascript
{
    keywords: ['ロゴデザイン', 'ロゴ'],
    answer: 'ロゴデザインにも対応しています。会社のイメージに合ったロゴを制作いたします。'
}
```

## 🎨 カスタマイズガイド

### メールアドレスの変更
- `index.html`の「Contact」セクションで、メールアドレスを変更できます
- Formspree経由なので、直接のメール設定は不要です

### SNSリンクの更新
- `index.html`と各ページ内のSNSリンク（Instagram, X, Pixiv等）をあなたのアカウントに置き換え

### 企業名や説明の変更
- `index.html`内の「solace」や説明文をあなたの情報に置き換え

### 画像の追加
- `images/`フォルダに画像を保存
- `sample1.jpg`, `sample2.jpg`, `sample3.jpg` をあなたの作品に置き換え

## 🆘 トラブルシューティング

### Q: フォームが送信できない
**A:** 
- Formspreeの測定IDが正しく設定されているか確認
- メールアドレスが登録済みか確認
- フォームの `method="POST"` が設定されているか確認

### Q: Google Analyticsが表示されない
**A:** 
- 測定IDが正しく設定されているか確認
- Google Analyticsダッシュボードで「リアルタイム」を確認
- 表示まで数分かかる場合があります

### Q: チャットボットが表示されない
**A:** 
- ブラウザのキャッシュをクリアしてリロード
- ブラウザコンソール（F12）でエラーがないか確認
- script.jsが正しく読み込まれているか確認

### Q: 画像が表示されない
**A:** 
- `images/`フォルダが正しく配置されているか確認
- ファイル名が正しいか確認（大文字小文字を区別）
- ホスティング後、キャッシュクリア後に確認
